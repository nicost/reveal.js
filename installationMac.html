Installation on Mac OS

If there is no "Anywhere" option (and ImageJ does not start), you can try to open a Terminal, type "sudo spctl --master-disable" (Write without ""), hit enter, provide password, and now hopefully find the Anywhere option.

If you would like to undo your change, type "sudo spctl --master-enable"
