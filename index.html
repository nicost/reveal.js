Measure electron conversion factor from Photon shot noise

DN = k * N      (1)
N - number of photons
k - electron conversion factor (ecf)
DN - digital number
√N - photon shot noise

If only noise is shot noise, then:
σ(DN) = √(DN)
and:
σ(DN) = k * √(N)      (2)

so: $$k = {\sigma{DN}}^2 / DN$$
